1. ĐẶT VẤN ĐỀ

Mục tiêu là viết chương trình tính $S = a! + b! + c!$ với $a, b, c$ là 3 số nguyên dương nhập từ bàn phím.

Quy trình thực hiện:

  1. Nhập $a, b, c > 0$.
  2. Tính $S = a! + b! + c!$.
  3. Xuất kết quả $S$.

Vấn đề lặp code (Code Redundancy)

Để giải quyết bài toán trên mà không dùng hàm, chúng ta phải lặp lại các đoạn lệnh sau nhiều lần:

A. 3 đoạn lệnh nhập $a, b, c > 0$:


// Nhập a > 0
int a;
do {
    cout << "Nhap mot so nguyen duong: ";
    cin >> a;
} while (a <= 0);

// (Tương tự cho b và c)

B. 3 đoạn lệnh tính $s1 = a!, s2 = b!, s3 = c!$:


//Tính s1 = a! = 1 * 2 * … * a
long s1 = 1; // Dùng long để tránh tràn số
for (int i = 2; i <= a ; i++)
    s1 = s1 * i;

// (Tương tự cho s2 = b! và s3 = c!)

Giải pháp

Giải pháp là viết 1 lần và sử dụng nhiều lần bằng cách sử dụng Hàm (Function).

Đoạn lệnh tổng quát hóa việc nhập số nguyên dương $n$:


int n;
do {
    cout << "Nhap mot so nguyen duong: ";
    cin >> n;
} while (n <= 0);

Đoạn lệnh tổng quát hóa việc tính giai thừa $s = n!$:


//Tính s = n! = 1 * 2 * … * n
long s = 1;
for (int i = 2; i <= n ; i++)
    s = s * i;

2. HÀM (FUNCTION) - KHÁI NIỆM VÀ CÚ PHÁP

1. Khái niệm

  • Hàm là một đoạn chương trình có tên, đầu vào và đầu ra.
  • Hàm có chức năng giải quyết một số vấn đề chuyên biệt cho chương trình chính.
  • Hàm được gọi nhiều lần với các tham số khác nhau.
  • Hàm được sử dụng khi có nhu cầu: Tái sử dụng, Sửa lỗi và Cải tiến.

2. Cú pháp

Cú pháp khai báo và định nghĩa một hàm như sau:


<kiểu trả về> <tên hàm>([<danh sách tham số>])
{
    <các câu lệnh>
    [return <giá trị>;]
}
Thành phần Ý nghĩa Chi tiết
<kiểu trả về> Kiểu dữ liệu mà hàm trả về Có thể là kiểu bất kỳ của C (char, int, long, float,…). Nếu không trả về, sử dụng từ khóa void.
<tên hàm> Tên để gọi hàm Theo quy tắc đặt tên định danh.
<danh sách tham số> Đầu vào hình thức của hàm Giống khai báo biến, cách nhau bằng dấu phẩy (,).
[return <giá trị>;] Lệnh trả về giá trị Trả về giá trị cho hàm. Bắt buộc nếu kiểu trả về không phải là void.

3. CÁC BƯỚC VIẾT HÀM

Khi viết hàm, cần xác định các thông tin sau:

  1. Tên hàm.
  2. Công việc hàm sẽ thực hiện.
  3. Các đầu vào (nếu có).
  4. Đầu ra (nếu có).
Tên hàm Đầu vào Đầu ra Công việc (Code minh họa)
Ví dụ 1: XuatTong
(Hàm không trả về)		 Hai số nguyên $x$ và $y$ Không có (void)

Tính và xuất tổng 2 số.


void XuatTong(int x, int y)
{
    int s;
    s = x + y;
    cout << x << " + " << y << " = " << s;
}
Ví dụ 2: TinhTong
(Hàm có trả về)		 Hai số nguyên $x$ và $y$ Một số nguyên (tổng)

Tính và trả về tổng 2 số.


int TinhTong(int x, int y)
{
    int s;
    s = x + y;
    return s;
}
Ví dụ 3: NhapXuatTong
(Không đầu vào, không trả về)		 Không có Không có (void)

Nhập và xuất tổng 2 số.


void NhapXuatTong()
{
    int x, y;
    cout << "Nhap 2 so nguyen: ";
    cin >> x >> y;
    cout << x << " + " << y << " = " << (x + y);
}

4. TẦM VỰC (PHẠM VI - SCOPE)

1. Khái niệm

Tầm vực là phạm vi hiệu quả của biến và hàm.

2. Phân loại Biến

  • Biến Toàn cục (Global):
    • Khai báo ngoài tất cả các hàm (kể cả hàm main).
    • Có tác dụng lên toàn bộ chương trình.
  • Biến Cục bộ (Local):
    • Khai báo trong hàm hoặc khối { }.
    • Chỉ có tác dụng trong bản thân hàm hoặc khối đó (kể cả khối con nó).
    • Biến cục bộ sẽ bị xóa khỏi bộ nhớ khi kết thúc khối khai báo nó.

3. Minh họa Tầm vực


#include <iostream>
using namespace std;

int a; // Biến toàn cục

int Ham1()
{
    int a1; // Biến cục bộ của Ham1
    a = 10; // OK: Có thể thấy biến toàn cục 'a'
    // a3 = 5; // LỖI: Không thể thấy 'a3' của main
    return 0;
}

int Ham2()
{
    int a2; // Biến cục bộ của Ham2
    {
        int a21; // Biến cục bộ của khối con
        a2 = 5; // OK: Khối con thấy 'a2' của khối cha
    }
    // a21 = 10; // LỖI: 'a21' không tồn tại ở đây
    return 0;
}

void main()
{
    int a3; // Biến cục bộ của main
    a = 20; // OK: main thấy biến toàn cục 'a'
    // a1 = 1; // LỖI: main không thấy 'a1' của Ham1
}

4. Một số lưu ý về khai báo hàm (Prototype)

Thông thường, người ta thường đặt phần tiêu đề hàm/nguyên mẫu hàm (prototype) trên hàm main và phần định nghĩa hàm dưới hàm main.


#include <iostream>
using namespace std;

// 1. Khai báo nguyên mẫu hàm (Prototype)
void XuatTong(int x, int y); 

void main()
{
    // 2. Lời gọi hàm
    XuatTong(10, 20); 
}

// 3. Định nghĩa hàm
void XuatTong(int x, int y) 
{
    cout << x << " cong " << y << " bang " << (x + y);
}

5. THAM SỐ VÀ LỜI GỌI HÀM

A. Các cách truyền đối số (Passing Arguments)

Trong một hàm, các tham số có thể được truyền theo nhiều cách khác nhau:

1. Truyền Giá trị (Call by Value)

  • Cách truyền: Truyền đối số cho hàm ở dạng giá trị (tạo ra một bản sao).
  • Đầu vào: Có thể truyền hằng, biến, biểu thức.
  • Mục đích: Được sử dụng khi không có nhu cầu thay đổi giá trị của biến gốc sau khi thực hiện hàm.

void TruyenGiaTri(int x)
{
    x++; // Chỉ thay đổi giá trị của bản sao cục bộ x
}
// int a = 5;
// TruyenGiaTri(a); // 'a' vẫn là 5

2. Truyền Địa chỉ (Call by Address - Con trỏ)

  • Cách truyền: Truyền đối số cho hàm ở dạng địa chỉ (dùng con trỏ *).
  • Đầu vào: Phải truyền địa chỉ của biến (ví dụ: &a).
  • Mục đích: Được sử dụng khi có nhu cầu thay đổi giá trị của biến gốc.

void TruyenDiaChi(int *x) // x là con trỏ
{
    (*x)++; // Thay đổi giá trị tại địa chỉ mà x trỏ tới
}
// int a = 5;
// TruyenDiaChi(&a); // 'a' bây giờ là 6

3. Truyền Tham chiếu (Call by Reference) (C++)

  • Cách truyền: Truyền tham chiếu đến biến gốc (dùng & trong khai báo).
  • Đầu vào: Truyền trực tiếp tên biến.
  • Mục đích: Tương tự con trỏ, dùng khi cần thay đổi giá trị biến gốc (cú pháp gọn hơn).

void TruyenThamChieu(int &x) // x là tham chiếu
{
    x++; // Thay đổi trực tiếp giá trị của biến gốc
}
// int a = 5;
// TruyenThamChieu(a); // 'a' bây giờ là 6

B. Lưu ý khi truyền đối số

Truyền Hỗn hợp: Trong một hàm, các tham số có thể truyền theo nhiều cách.


void HonHop(int x, int &y)
{
    x++; // Truyền giá trị (biến gốc không đổi)
    y++; // Truyền tham chiếu (biến gốc thay đổi)
}

Sử dụng Tham chiếu để Trả về Giá trị: Sử dụng tham chiếu là một cách hiệu quả để "trả về" nhiều giá trị.

Trả về bằng return "Trả về" bằng Tham chiếu & "Trả về" nhiều giá trị

    return x + y;

C. Lời gọi hàm (Function Call)

  • Cách thực hiện: Gọi tên của hàm đồng thời truyền các đối số (hằng, biến, biểu thức) cho các tham số theo đúng thứ tự.
  • Cú pháp: <tên hàm> (<đối số 1>,… , <đối số n>);.

Ví dụ Lời gọi: (Giả sử int n = 9;)

Ví dụ Hoán vị (HoanVi):

Sử dụng truyền tham chiếu để thay đổi giá trị của hai biến $a$ và $b$ gốc.


// Khai báo hàm dùng tham chiếu
void HoanVi(int &a, int &b)
{
    int tam = a;
    a = b;
    b = tam;
}

void main()
{
    int x = 2912, y = 1706;
    cout << "Truoc: x=" << x << ", y=" << y << endl;
    
    HoanVi(x, y); // Lời gọi
    
    cout << "Sau: x=" << x << ", y=" << y << endl;
    // x bây giờ là 1706, y là 2912
}

6. ĐỆ QUY (RECURSION)

1. Khái niệm

  • Một chương trình con có thể gọi một chương trình con khác.
  • Nếu chương trình con gọi chính nó thì được gọi là sự đệ quy.
  • Quy tắc bắt buộc: Số lần gọi này phải có giới hạn (phải có điểm dừng).

2. Ví dụ: Tính Giai Thừa $S(n) = n!$

Định nghĩa giai thừa: $S(n) = 1 \times 2 \times … \times (n-1) \times n$.

Ta nhận thấy mối quan hệ đệ quy: $S(n) = S(n-1) \times n$.

Thay vì tính $S(n)$, ta sẽ đi tính $S(n-1)$, rồi $S(n-2)$, …, cho đến khi đạt được điểm dừng là $S(1)$, và cuối cùng là $S(0) = 1$.

3. Thuật toán Đệ quy Giai Thừa

Đây là thuật toán quan trọng:

Phiên bản 1:


long GiaiThua(int n)
{
    if (n == 0) // Điểm dừng
        return 1;
    else
        return GiaiThua(n - 1) * n; // Lời gọi đệ quy
}

Phiên bản 2 (Biến thể):


long GiaiThua(int n)
{
    if (n > 0)
        return GiaiThua(n - 1) * n;
    else // Điểm dừng khi n <= 0
        return 1;
}

7. BÀI TẬP ỨNG DỤNG

1. Bài tập Chương Câu lệnh Điều kiện và Rẽ nhánh

  1. Viết hàm đổi một ký tự hoa sang ký tự thường.
  2. Viết thủ tục (hàm void) giải phương trình bậc nhất.
  3. Viết thủ tục giải phương trình bậc hai.
  4. Viết hàm trả về giá trị nhỏ nhất của 4 số nguyên.
  5. Viết thủ tục hoán vị hai số nguyên (dùng tham chiếu).
  6. Viết thủ tục sắp xếp 4 số nguyên tăng dần (dùng hàm hoán vị).

2. Bài tập Chương Câu lệnh Lặp (Phân tích số $n$)

Hàm nhận vào một số nguyên dương $n$ và thực hiện:

  1. Trả về số đảo của số đó.
  2. Kiểm tra có phải là số đối xứng (Trả về true/false).
  3. Kiểm tra có phải là số chính phương.
  4. Kiểm tra có phải là số nguyên tố.
  5. Trả về tổng các chữ số lẻ.
  6. Trả về tổng các chữ số nguyên tố.
  7. Trả về tổng các chữ số chính phương.

3. Bài tập Chương Câu lệnh Lặp (Hàm tính tổng S)

Hàm nhận vào một số nguyên dương $n$ và thực hiện tính các tổng sau:

  1. $S = 1 + 2 + … + n$
  2. $S = 1^2 + 2^2 + … + n^2$
  3. $S = 1 + 1/2 + … + 1/n$
  4. $S = 1 \times 2 \times … \times n$ (Giai thừa $n!$)
  5. $S = 1! + 2! + … + n!$ (Gợi ý: dùng lại hàm tính giai thừa ở câu 3d)

4. Các bài tập khác

  • Viết hàm trả về USCLN của 2 số nguyên.
  • Viết thủ tục in ra $n$ phần tử của dãy Fibonacy.
