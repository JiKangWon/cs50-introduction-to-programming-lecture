1. KHÁI NIỆM

1.1. Định nghĩa

Kiểu char trong lập trình chỉ chứa được một ký tự. Để lưu trữ một chuỗi (nhiều ký tự), ta sử dụng mảng (một chiều) các ký tự.

1.2. Ký tự kết thúc chuỗi và Độ dài

  • Chuỗi ký tự bắt buộc phải kết thúc bằng ký tự '\0' (null).
  • Độ dài chuỗi (số ký tự tối đa có thể lưu trữ) được xác định như sau:
$$ \text{Độ dài chuỗi} = \text{kích thước mảng} – 1 $$

1.3. Ví dụ Khai báo


// Chuỗi này có độ dài tối đa là 29 ký tự.
// (Cần 30 ô nhớ: 29 cho ký tự + 1 cho '\0')
char hoten[30]; 

// Chuỗi này có độ dài tối đa là 8 ký tự.
// (Cần 9 ô nhớ: 8 cho ký tự + 1 cho '\0')
char ngaysinh[9];

2. KHỞI TẠO CHUỖI

Chuỗi ký tự có thể được khởi tạo tương tự như khởi tạo mảng thông thường, với hai phương pháp chính: Khởi tạo với độ dài cụ thể, hoặc Tự xác định độ dài.

2.1. Các phương thức khởi tạo

A. Khởi tạo tường minh bằng mảng ký tự (cần '\0')


// Khai báo có kích thước
char s[7] = {'T', 'H', 'C', 'S', ' ', 'A', '\0'};

// Tự xác định kích thước (sẽ là 7)
char s[] = {'T', 'H', 'C', 'S', ' ', 'A', '\0'};

B. Khởi tạo bằng hằng chuỗi (tự động thêm '\0')

Khi sử dụng hằng chuỗi (dấu ngoặc kép ""), ký tự '\0' sẽ được tự động thêm vào cuối chuỗi.


// Khai báo có kích thước
char s[30] = "THCS A"; // Tự động thêm '\0'

// Tự xác định kích thước (sẽ là 7)
char s[] = "THCS A"; // Tự động thêm '\0'

Biểu diễn lưu trữ:

Đối với char s[] = "THCS A";, chuỗi được lưu trữ như sau (7 ô nhớ):

Vị trí 0 1 2 3 4 5 6
Ký tự ‘T’ ‘H’ ‘C’ ‘S’ ‘ ’ ‘A’ ‘\0’

3. CÁC THAO TÁC TRÊN CHUỖI KÝ TỰ

3.1. Xuất chuỗi

Sử dụng cout để xuất chuỗi ra màn hình.


#include <iostream>
using namespace std;

void main()
{
    char monhoc[] = "Tin hoc co so A";
    cout << "Ten mon hoc: " << monhoc;
    // Output: Ten mon hoc: Tin hoc co so A
}

3.2. Nhập chuỗi

3.2.1. Sử dụng cin

  • Nguyên tắc: Chỉ nhận các ký tự từ bàn phím đến khi gặp ký tự khoảng trắng hoặc ký tự xuống dòng.
  • Kết quả: Chuỗi nhận được không bao gồm ký tự khoảng trắng và xuống dòng.

char monhoc[50]; // Phải có kích thước
cout << "Nhap mot chuoi: ";
cin >> monhoc;

// Khi người dùng nhập: Tin hoc co so A
// Chuoi nhan duoc la: "Tin"

3.2.2. Sử dụng hàm cin.getline

  • Nguyên tắc: Nhận các ký tự từ bàn phím đến khi gặp ký tự xuống dòng.
  • Kết quả: Chuỗi nhận được là toàn bộ nội dung người dùng nhập (trừ ký tự xuống dòng).
  • Cú pháp: cin.getline(tên_chuỗi, kích thước_tối_đa);

char monhoc[50];
cout << "Nhap mot chuoi : ";
cin.getline(monhoc, 50);

// Khi người dùng nhập: Tin hoc co so A
// Chuoi nhan duoc la: "Tin hoc co so A"

3.3. Một số hàm thao tác trên chuỗi theo phong cách C

Các hàm này thuộc thư viện <string.h>.

Tên Hàm Mô tả chi tiết (Algorithm) Cú pháp/Công thức
strlen Tính độ dài chuỗi $s$ (không tính '\0'). size_t strlen(const char *s)
strcpy Sao chép chuỗi src sang chuỗi dest.
Lưu ý: dest phải đủ lớn.		 char* strcpy(char *dest, const char *src)
strdup Tạo bản sao của chuỗi $s$. Hàm tự tạo vùng nhớ.
Lưu ý: Phải tự hủy vùng nhớ này.		 char* strdup(const char *s)
strlwr Chuyển chuỗi $s$ thành chuỗi thường. char* strlwr(char *s)
strupr Chuyển chuỗi $s$ thành chuỗi IN HOA. char* strupr(char *s)
strrev Đảo ngược thứ tự các ký tự trong chuỗi $s$. char* strrev(char *s)
strcmp So sánh $s1$ và $s2$ (phân biệt hoa thường).
Trả về: < 0 ($s1 < s2$); 0 ($s1 == s2$); > 0 ($s1 > s2$).		 int strcmp(const char *s1, const char *s2)
stricmp So sánh $s1$ và $s2$ (KHÔNG phân biệt hoa thường).
Trả về: < 0 ($s1 < s2$); 0 ($s1 == s2$); > 0 ($s1 > s2$).		 int stricmp(const char *s1, const char *s2)
strcat Nối chuỗi src vào sau chuỗi dest.
Lưu ý: dest phải đủ chứa kết quả.		 char* strcat(char *dest, const char *src)
strstr Tìm vị trí xuất hiện đầu tiên của $s2$ trong $s1$.
Trả về con trỏ đến vị trí tìm thấy hoặc NULL.		 char* strstr(const char *s1, const char *s2)

3.4. Xử lý chuỗi bằng C++ STL (Standard Template Library)

Thư viện chuẩn STL của C++ (<string>) hỗ trợ kiểu string với các phép toán và phương thức tiện lợi.

3.4.1. Các phép toán và phương thức cơ bản

Thao tác Cú pháp/Phương thức Mô tả chi tiết
Ghép chuỗi Phép cộng + Dùng để ghép hai chuỗi hoặc ghép một ký tự vào chuỗi.
So sánh ==, !=, >, >=, <, <= So sánh theo thứ tự từ điển.
Duyệt ký tự s.length(), toán tử [] Nếu $s$ là string, $s[i]$ là ký tự thứ $i$ ($0 \leq i < s.length()$).
Gán Phép gán = Dùng để gán biến string bằng một hằng chuỗi.

3.4.2. Các phương thức nâng cao

Lấy chuỗi con:


// Cú pháp: substr(int pos, int nchar)
// Mô tả: Trả về chuỗi con gồm nchar ký tự của chuỗi str kể từ vị trí pos.
string s = "Hello World";
string sub = s.substr(6, 5); // sub = "World"

Chèn thêm ký tự hay chuỗi (insert):


string str = "to be question";
// Chèn chuỗi C-style (char*)
str.insert(6, "the "); // str = "to be the question"

// Chèn chuỗi kiểu string
string s2 = "or not ";
str.insert(6, s2); // str = "to be or not the question"

// Chèn n lần ký tự ch
str.insert(10, 3, '!'); // str = "to be or not!!! the question"

Xóa chuỗi (erase):


// Cú pháp: erase(int pos, int n)
// Mô tả: Xóa n ký tự của chuỗi str kể từ vị trí pos.
string str = "This is an example";
str.erase(10, 3); // Xóa "exa", str = "This is an mple"
str.erase(7); // Xóa từ vị trí 7 đến hết
// str = "This is"

Tìm kiếm chuỗi (find):


// Cú pháp: find(chuoi_can_tim, int pos = 0)
// Mô tả: Tìm chuỗi con kể từ vị trí pos. Trả về vị trí tìm thấy
// hoặc string::npos (giá trị đặc biệt) nếu không tìm thấy.

string str = "There are two needles";
int found = str.find("needle"); // found = 14

found = str.find("example"); // found = string::npos

4. BÀI TẬP ỨNG DỤNG

Danh sách các bài tập được đề xuất để luyện tập các thao tác trên chuỗi:

  1. Bài 1: Viết hàm nhận vào một chuỗi và trả về chuỗi tương ứng (giữ nguyên chuỗi đầu vào) với các yêu cầu sau:
    • Các ký tự thành ký tự thường (giống strlwr).
    • Các ký tự thành ký tự hoa (giống strupr).
    • Các ký tự đầu tiên của mỗi từ thành ký tự hoa.
    • Chuẩn hóa chuỗi (xóa khoảng trắng thừa ở đầu, cuối và giữa các từ).
  2. Bài 2: Viết hàm nhận vào một chuỗi $s$ và trả về chuỗi tương ứng sau khi xóa các khoảng trắng.
  3. Bài 3: Viết hàm nhận vào một chuỗi $s$ và đếm xem có bao nhiêu từ trong chuỗi đó.
  4. Bài 4: Viết chương trình nhập vào họ tên của một sinh viên, cho biết họ tên sinh viên đó có lót chữ “thị” hay không.
  5. Bài 5: Viết chương trình tách phần tên trong một chuỗi chứa họ và tên Việt Nam. (Ví dụ: Chuỗi "Nguyen Van Ba" $\to$ "Ba").
  6. Bài 6: Viết hàm tìm từ có chiều dài lớn nhất và xuất ra màn hình từ đó và độ dài tương ứng.
  7. Bài 7: Viết hàm trích ra $n$ ký tự đầu tiên/cuối cùng/bắt đầu tại vị trí $pos$ của chuỗi $s$ cho trước.
  8. Bài 8: Viết hàm kiểm tra một chuỗi có đối xứng hay không. (Ví dụ: Chuỗi "MADAM" là đối xứng).
  9. Bài 9: Viết hàm nhận vào một chuỗi $s$ và xuất các từ trên các dòng liên tiếp.
  10. Bài 10: Viết chương trình cho phép nhập vào danh sách có $n$ họ tên sinh viên:
    • Đếm xem trong danh sách có bao nhiêu sinh viên có họ là họ “nguyễn”.
    • Sắp xếp danh sách tăng dần theo họ.
